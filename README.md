# webCrmClient
webCRM REST API library and test client

The .model classes are generated by swagger-codegen with https://api.webcrm.com/documentation/index.html. 

# Installation and Settings
- Log into webCRM as an admin 
  * generate an API key for a system user.
  * allow the above system user to access Organisations, Persons, and Relations
- Right click the WebCrm.Rest.Net.
  * Click Properties > Settings.
  * Insert your API key.
- Run the WebCrm.Rest.UI for a demo
